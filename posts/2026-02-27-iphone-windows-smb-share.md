# Windows 與 iPhone 檔案互傳：免軟體、內網共享資料夾設定指南

要在 Windows 與 iPhone 之間互傳檔案，除了使用 iTunes、iCloud 或第三方雲端服務外，其實有一種更直接、更私密的方法：**建立 Windows 共用資料夾**。只要兩台裝置在同一個內網下，就能像存取隨身碟一樣輕鬆傳輸。

![Windows 與 iPhone 互傳檔案](/static/images/smb-share-hero.png)

## 🛠️ Windows 端設定步驟

為了確保安全性，建議建立一個專用的本機帳號來存取資料夾，並嚴格控管權限。

### 步驟 1：建立專用帳號
使用 **管理員身分** 開啟 CMD，輸入以下指令新增一個本機使用者（以帳號 `test`、密碼 `123456` 為例）：

```bash
net user /add test 123456

```

### 步驟 2：設定資料夾權限

找到您想分享的資料夾，右鍵點選「內容」→「安全性」→「進階」：

1. **停用繼承**：點選「停用繼承」，並將此物件的繼承權限移除。
2. **清理權限項目**：除了 **SYSTEM**、**Domain Admin** (如有) 以及 **Administrators** 之外，移除其他所有帳號。
3. **新增存取帳號**：點選「新增」，選擇剛才建立的 **test** 帳號，並給予讀取/寫入權限。

### 步驟 3：開啟資料夾共用

在資料夾內容的「共用」分頁中，點選「共用」，將 **test** 加入名單並設定為「讀取/寫入」。

## 📱 iPhone 端連線步驟

設定完 PC 後，請確保 iPhone 已連線至同一個 Wi-Fi 網路。

1. 打開 iPhone 內建的 **「檔案」** App。
2. 點選右上角的「...」圖示，選擇 **「連接伺服器」**。
3. 輸入電腦的內網 IP 位址（例如：`192.168.1.86`）。
4. 選擇「註冊使用者」，輸入剛才設定的帳號 `test` 與密碼 `123456`。

## 💡 總結

透過這種方式，您不需要安裝任何輔助軟體或上傳到第三方雲端，就能在手機與電腦間快速拖放檔案。這不僅保護隱私，在傳輸大型影片或大量圖片時也更具效率。
